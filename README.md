# voca-web
블로그 프로젝트

* 프로젝트 실행 방법<br>
`json-server --watch ./src/db/data.json --port 3001`<br>
`npm run start`

* DB는 npm에서 제공하는 json-server를 이용하였습니다.<br>
https://www.npmjs.com/package/json-server<br><br>
